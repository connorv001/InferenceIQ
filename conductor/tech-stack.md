# Technology Stack

## Core Language & Runtime
- **Language:** Python 3.10+
  - *Rationale:* Standard for ML/AI engineering, rich ecosystem for data analysis (`pandas`) and API interactions.

## Data Collection & Instrumentation
- **LLM SDKs:**
  - `openai`: For tracking OpenAI API usage.
  - `anthropic`: For tracking Anthropic API usage.
  - `boto3`: For AWS Bedrock and CloudWatch integration.
- **Infrastructure Integration:**
  - `kubectl`: For querying Kubernetes pod resource usage.
  - `subprocess`: Python standard library to execute shell commands for infra checks.

## Data Processing & Analysis
- **Data Manipulation:** `pandas`
  - *Usage:* Ingesting JSONL logs, calculating aggregates, and performing cost analysis (grouping by model, customer, outcome).
- **Data Storage (MVP):** Local JSONL files (`genai_costs.jsonl`)
  - *Rationale:* Simple, append-only, human-readable, and requires no database setup for the POC phase.

## Dashboard & Visualization
- **Visualization Engine:** `matplotlib` or `plotly` (via Python)
  - *Usage:* Generating static charts (PNG/SVG) for reports or interactive plots.
- **Dashboard Format:** Single-file HTML Report
  - *Usage:* Generated by Python script, embedding CSS/JS for a portable "Executive Dashboard" that requires no hosting server.
  - *Alternative:* `Streamlit` (if interactive filtering is strictly required, but HTML report is more portable for email).

## Compliance & Monitoring
- **Logging Standard:** JSON Structured Logging
  - *Rationale:* Machine-readable, easy to parse for audit trails (RBI compliance).
